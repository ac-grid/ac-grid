## 📍 项目愿景

AC Grid 致力于成为 **基于 Web Components 标准的高性能数据表格组件库**，作为 ag-Grid 的开源替代方案。我们的目标是提供：

- 🎯 **框架无关**: 可在任何前端框架中使用
- ⚡ **高性能**: 支持 100K+ 行数据
- 🎨 **易定制**: 灵活的主题和样式系统
- 📦 **轻量级**: 最小化包体积
- 🔧 **开发友好**: 完整的 TypeScript 支持和文档

## 🤝 如何贡献

我们欢迎社区贡献！以下是参与方式：

### 1. 选择一个任务
查看 [RFC 索引](./docs/rfc/README.md) 选择一个待实施的功能。

### 2. 编写或审查 RFC
- 如果 RFC 不存在，使用模板创建
- 如果 RFC 存在，审查并提供反馈

### 3. 实施功能
按照 RFC 中的实施步骤开发功能。

### 4. 提交 PR
确保：
- ✅ 100% 测试覆盖率
- ✅ 零 Lint 错误
- ✅ 完整的文档
- ✅ Storybook 示例

### 5. 更新路线图
功能完成后，在本文件中更新状态。

**详细指南**: 查看 [GETTING-STARTED.md](./docs/rfc/GETTING-STARTED.md)
